# Projeto LeiloesTDSat

## Sobre o Projeto
Este repositório contém o sistema informatizado desenvolvido para gestão de casas de leilões. O objetivo é automatizar as rotinas de vendas e lances, garantindo a segurança e integridade dos dados, conforme as práticas de qualidade de software exigidas.

## Tecnologias Utilizadas
* **Linguagem:** Java
* **Banco de Dados:** MySQL
* **Versionamento:** Git e GitHub
